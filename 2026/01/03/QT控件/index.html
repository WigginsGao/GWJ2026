geometry

иЎЁзӨәеҪ“еүҚжҺ§д»¶зҡ„дҪҚзҪ®е’Ңе°әеҜёпјҡxгҖҒyгҖҒwidgetгҖҒhight

еҜ№ дәҺQtзҡ„еқҗж Үзі»,дёҚиҰҒеҝҳи®°жҳҜвјҖдёӘвҖқе·Ұвјҝеқҗж Үзі»вҖқ.е…¶дёӯеқҗж Үзі»зҡ„еҺҹзӮ№жҳҜеҪ“еүҚе…ғзҙ зҡ„вҪ—е…ғзҙ зҡ„е·ҰдёҠв»Ҷ

API

  • geometry()пјҡиҺ·еҸ–еҲ°жҺ§д»¶зҡ„дҪҚзҪ®е’Ңе°әвјЁ.иҝ”еӣһз»“жһңжҳҜвјҖдёӘQRect,еҢ…еҗ«дәҶx,y,width,height.е…¶дёӯx,yжҳҜе·ҰдёҠв»Ҷзҡ„еқҗж Ү.
  • setGeometry(QRect)гҖҒsetGeometry(int x, int y, int width, int height)пјҡи®ҫзҪ®жҺ§д»¶зҡ„дҪҚзҪ®е’Ңе°әвјЁ.еҸҜд»ҘзӣҙжҺҘи®ҫзҪ®вјҖдёӘQRect,д№ҹеҸҜд»ҘеҲҶеӣӣдёӘеұһжҖ§еҚ•зӢ¬и®ҫзҪ®.

д»Јз ҒзӨәдҫӢ

йҖҡиҝҮupгҖҒdownгҖҒleftгҖҒrightжҢүй’®жҺ§еҲ¶targetжҢүй’®дёҠдёӢе·ҰеҸіз§»еҠЁ

windowframeзҡ„еҪұе“Қ

еҰӮжһңwidgetдҪңдёәвјҖдёӘзӘ—вјқ(еёҰжңүж Үйўҳж Ҹ,жңҖвј©еҢ–,жңҖвјӨеҢ–,е…ій—ӯжҢүй’®),йӮЈд№ҲеңЁи®Ўз®—е°әвјЁе’Ңеқҗж Үзҡ„ ж—¶еҖҷе°ұжңүдёӨз§Қз®—жі•.еҢ…еҗ«windowframeе’ҢдёҚеҢ…еҗ«windowframe.

  • е…¶дёӯx(),y(), frameGeometry(), pos(), move() йғҪжҳҜжҢүз…§еҢ…еҗ«windowframeзҡ„вҪ…ејҸжқҘи®Ўз®—зҡ„
  • е…¶дёӯgeometry(),width(), height(), rect(), size() еҲҷжҳҜжҢүз…§дёҚеҢ…еҗ«windowframeзҡ„вҪ…ејҸжқҘи®Ўз®—зҡ„.
API иҜҙжҳҺ
x() иҺ·еҸ–жЁӘеқҗж ҮпјҲи®Ўз®—ж—¶ еҢ…еҗ« window frameпјү
y() иҺ·еҸ–зәөеқҗж ҮпјҲи®Ўз®—ж—¶ еҢ…еҗ« window frameпјү
pos() иҝ”еӣһ QPoint еҜ№иұЎпјҢеҗ« x(), y(), setX(), setY() зӯүж–№жі•пјҲи®Ўз®—ж—¶ еҢ…еҗ« window frameпјү
frameSize() иҝ”еӣһ QSize еҜ№иұЎпјҢеҗ« width(), height(), setWidth(), setHeight() зӯүж–№жі•пјҲи®Ўз®—ж—¶ еҢ…еҗ« window frameпјү
frameGeometry() иҝ”еӣһ QRect еҜ№иұЎпјҢеҸҜиҺ·еҸ– x, y, width, heightпјҲи®Ўз®—ж—¶ еҢ…еҗ« window frameпјү
width() иҺ·еҸ–е®ҪеәҰпјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү
height() иҺ·еҸ–й«ҳеәҰпјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү
size() иҝ”еӣһ QSize еҜ№иұЎпјҢеҗ« width(), height(), setWidth(), setHeight() зӯүж–№жі•пјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү
rect() иҝ”еӣһ QRect еҜ№иұЎпјҢеҸҜиҺ·еҸ–е№¶и®ҫзҪ® x, y, width, heightпјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү
geometry() иҝ”еӣһ QRect еҜ№иұЎпјҢеҸҜиҺ·еҸ– x, y, width, heightпјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү
setGeometry() зӣҙжҺҘи®ҫзҪ®зӘ—еҸЈзҡ„дҪҚзҪ®е’Ңе°әеҜёпјҢеҸҜдј е…Ҙ x, y, width, height жҲ– QRect еҜ№иұЎпјҲи®Ўз®—ж—¶ дёҚеҢ…еҗ« window frameпјү

д»Јз ҒзӨәдҫӢ

еҶҷеңЁ Widget зҡ„жһ„йҖ еҮҪж•°йҮҢпјҢжӯӨж—¶зӘ—еҸЈпјҲWidgetпјүеҲҡеҲҡиў« new еҮәжқҘпјҢиҝҳжІЎжңүиў«зӘ—еҸЈз®ЎзҗҶеҷЁиЈ…йҘ°пјҲеҚіиҝҳжІЎеҠ дёҠж Үйўҳж ҸгҖҒиҫ№жЎҶзӯү window frameпјүгҖӮ

  • geometry() вҖ”вҖ” иҝ”еӣһзҡ„жҳҜ е®ўжҲ·еҢә зҡ„еҮ дҪ•пјҲдёҚеҗ« window frameпјүгҖӮ
  • frameGeometry() вҖ”вҖ” зҗҶи®әдёҠеә”иҜҘиҝ”еӣһвҖңе®ўжҲ·еҢә + window frameвҖқзҡ„еҮ дҪ•пјҢдҪҶжӯӨеҲ» frame е°ҡдёҚеӯҳеңЁпјҢжүҖд»ҘдёӨиҖ…еҫ—еҲ°зҡ„ж•°еҖје®Ңе…ЁдёҖж ·гҖӮ

жү“еҚ°з»“жһңпјҡ

дҪҶжҳҜеңЁзӮ№еҮ»жҢүй’®ж—¶,жү“еҚ°зҡ„geometryе’ҢframeGeometryеҲҷеӯҳеңЁе·®ејӮпјҡ

жү“еҚ°з»“жһңпјҡ

windowTitle

  • windowTitle()пјҡиҺ·еҸ–жҺ§д»¶зҡ„зӘ—еҸЈж ҮйўҳгҖӮ
  • setWindowTitle(const QString &title)пјҡи®ҫзҪ®жҺ§д»¶зҡ„зӘ—еҸЈж ҮйўҳгҖӮ

еҰӮжһңжҳҜйЎ¶еұӮwidget(зӢ¬вҪҙзӘ—вјқ),иҝҷдёӘж“ҚдҪңжүҚдјҡжңүж•ҲпјҢеҰӮжһңжҳҜвјҰwidget,иҝҷдёӘж“ҚдҪңвҪҶд»»дҪ•ж•ҲжһңгҖӮ

windowIcon

  • windowIcon()пјҡиҺ·еҸ–жҺ§д»¶зҡ„зӘ—еҸЈеӣҫж ҮпјҢиҝ”еӣһ QIcon еҜ№иұЎгҖӮ
  • setWindowIcon(const QIcon &icon)пјҡи®ҫзҪ®жҺ§д»¶зҡ„зӘ—еҸЈеӣҫж ҮгҖӮ

еҗҢwindowTitle,дёҠиҝ°ж“ҚдҪңд»…й’ҲеҜ№йЎ¶еұӮwidgetжңүж•ҲгҖӮ

QIcon жң¬иә«еҫҲиҪ»йҮҸпјҢеҸӘжҳҜвҖңеӣҫж ҮжҸҸиҝ°вҖқпјҢзңҹжӯЈз»ҳеӣҫж•°жҚ®з”ұ Qt еҶ…йғЁзј“еӯҳпјӣе®ғж—ўдёҚеҸӮдёҺеҜ№иұЎж ‘пјҢд№ҹж— жі•и®ҫзҪ®зҲ¶еҜ№иұЎгҖӮ
еӣ жӯӨпјҡ

  • еҸӘиҰҒеӣҫж Үе·Із»ҸйҖҡиҝҮ setWindowIcon() и®ҫзҪ®еҲ°жҺ§д»¶пјҢQIcon еҜ№иұЎжң¬иә«йҡҸеҗҺй”ҖжҜҒд№ҹдёҚдјҡеҪұе“Қз•ҢйқўжҳҫзӨәгҖӮ
  • з”Ёж ҲеҜ№иұЎгҖҒдёҙж—¶еҜ№иұЎжҲ–зӣҙжҺҘеҶҷ setWindowIcon(QIcon(":/res/icon.png")) йғҪеҸҜд»ҘпјҢж— йңҖеҲ»ж„Ҹ newгҖӮ

д»Јз ҒзӨәдҫӢ

  1. и®ҫзҪ®зӘ—вјқеӣҫж ҮвҖ“з»қеҜ№и·Ҝеҫ„

е®һйҷ…ејҖеҸ‘дёӯ,жҲ‘д»¬вјҖиҲ¬дёҚдјҡеңЁд»Јз ҒдёӯйҖҡиҝҮз»қеҜ№и·Ҝеҫ„еј•вјҠеӣҫвҪҡ.

еӣ дёәжҲ‘д»¬вҪҶжі•дҝқиҜҒзЁӢеәҸеҸ‘еёғеҗҺ,вҪӨ вјҫзҡ„з”өи„‘дёҠд№ҹжңүеҗҢж ·зҡ„и·Ҝеҫ„.

еҰӮжһңдҪҝвҪӨзӣёеҜ№и·Ҝеҫ„,еҲҷйңҖиҰҒзЎ®дҝқд»Јз Ғдёӯзҡ„зӣёеҜ№и·Ҝеҫ„еҶҷжі•е’ҢеӣҫвҪҡе®һйҷ…жүҖеңЁзҡ„и·Ҝеҫ„еҢ№й…Қ(вҪҗеҰӮд»Јз Ғ дёӯеҶҷдҪңвҖқ./image/rose.jpgвҖқ, е°ұйңҖиҰҒеңЁеҪ“еүҚвјҜдҪңвҪ¬еҪ•дёӯеҲӣе»әimageвҪ¬еҪ•,е№¶жҠҠrose.jpgж”ҫиҝӣ еҺ»).

  1. дҪҝз”ЁQRCжңәеҲ¶

QRCжңәеҲ¶зҡ„жң¬иҙЁжҳҜжҠҠеӣҫзүҮзҡ„дәҢиҝӣеҲ¶ж•°жҚ®иҪ¬жҲҗC++д»Јз ҒпјҢжңҖз»ҲдјҡеңЁд»Јз ҒдёӯзңӢеҲ°дёҖдёӘеҫҲеӨ§зҡ„charж•°з»„пјҲеңЁеҶ…еӯҳдёӯпјүпјҢйҮҢйқўеӯҳж”ҫзҡ„жҳҜдәҢиҝӣеҲ¶ж•°жҚ®гҖӮ

дёәдәҶеңЁQTд»Јз ҒдёӯиғҪеӨҹи®ҝй—®д»Јз ҒпјҢеӣ жӯӨQTе°ұиҮӘе·ұеҲӣе»әдёҖдёӘиҷҡжӢҹзӣ®еҪ•гҖӮ

жіЁж„ҸпјҡеңЁеҜје…ҘеӣҫзүҮзҡ„ж—¶еҖҷйңҖиҰҒзЎ®дҝқеҜје…Ҙзҡ„еӣҫзүҮеңЁresource.qrcж–Үд»¶зҡ„еҗҢзә§зӣ®еҪ•жҲ–иҖ…еҗҢзә§зӣ®еҪ•зҡ„еӯҗзӣ®еҪ•дёӯгҖӮ

  • дјҳзӮ№:зЎ®дҝқдәҶеӣҫвҪҡ,еӯ—дҪ“,еЈ°вҫізӯүиө„жәҗиғҪеӨҹзңҹжӯЈеҒҡеҲ°вҖқвҪ¬еҪ•вҪҶе…івҖң,вҪҶи®әеҰӮдҪ•йғҪдёҚдјҡеҮәзҺ°иө„жәҗдёўеӨұзҡ„жғ…еҶөгҖӮ

  • зјәзӮ№:дёҚйҖӮеҗҲз®ЎзҗҶдҪ“з§ҜвјӨзҡ„иө„жәҗ.еҰӮжһңиө„жәҗвҪҗиҫғвјӨ(вҪҗеҰӮжҳҜвјҸдёӘMBзҡ„вҪӮд»¶),жҲ–иҖ…иө„жәҗзү№еҲ«еӨҡ, вҪЈжҲҗзҡ„жңҖз»Ҳзҡ„exeдҪ“з§Ҝе°ұдјҡвҪҗиҫғвјӨ,зЁӢеәҸиҝҗвҫҸж¶ҲиҖ—зҡ„еҶ…еӯҳд№ҹдјҡеўһвјӨ,зЁӢеәҸзј–иҜ‘зҡ„ж—¶й—ҙд№ҹдјҡжҳҫи‘— еўһеҠ гҖӮ

жіЁж„ҸпјҡC++ зҡ„ float йҒөеҫӘ IEEE 754 ж ҮеҮҶпјҢиҝҗз®—еӯҳеңЁзІҫеәҰиҜҜе·®гҖӮ
еӣ жӯӨ 1 - 0.1 зҡ„з»“жһңе№¶дёҚзІҫзЎ®зӯүдәҺ 0.9пјҢеҸҜиғҪжҳҫзӨәдёә 0.899 999 976вҖҰ е·ҰеҸігҖӮ
еҜ№йҖҸжҳҺеәҰжҲ–е…¶д»–йңҖиҰҒзІҫзЎ®жҜ”иҫғзҡ„еңәжҷҜпјҢиҜ·йҒҝе…ҚзӣҙжҺҘ == еҲӨж–ӯпјҢж”№з”ЁеҢәй—ҙжҲ–иҜҜе·®иҢғеӣҙжҜ”иҫғгҖӮ

еӣ жӯӨпјҢеңЁи®Ўз®—жө®зӮ№ж•°иҝҗз®—ж—¶дёҖе®ҡиҰҒжіЁж„Ҹпјҡ

  • дёҚеҮҶпјҡдәҢиҝӣеҲ¶ж— жі•зІҫзЎ®иЎЁзӨәеҚҒиҝӣеҲ¶е°Ҹж•°пјҢ0.1 еӯҳзҡ„жҳҜвҖңзәҰзӯүдәҺ 0.1 зҡ„ж— з©·зә§ж•°жҲӘж–ӯеҖјвҖқгҖӮ
    дәҺжҳҜ0.1 + 0.2 == 0.3 вҶ’ falseпјҢе®һйҷ…еҫ—еҲ° 0.300 000 000 000 000 04гҖӮ

йҖҡиҝҮ setFocusPolicy() еҸҜеҶіе®ҡжҺ§д»¶иғҪеҗҰиў«йј ж ҮзӮ№еҮ»жҲ– Tab й”®йҖүдёӯпјҢеҜ№иҫ“е…ҘжЎҶгҖҒеҚ•йҖүжЎҶгҖҒеӨҚйҖүжЎҶзӯүе°Өе…¶йҮҚиҰҒгҖӮ

  • focusPolicy()
    иҝ”еӣһеҪ“еүҚжҺ§д»¶зҡ„з„ҰзӮ№зӯ–з•ҘпјҢзұ»еһӢдёә Qt::FocusPolicyгҖӮ

    • Qt::NoFocus вҖ”вҖ” ж— жі•йҖҡиҝҮд»»дҪ•ж–№ејҸиҺ·еҫ—з„ҰзӮ№гҖӮ
    • Qt::TabFocus вҖ”вҖ” д»… Tab й”® еҸҜиҒҡз„ҰгҖӮ
    • Qt::ClickFocus вҖ”вҖ” д»… йј ж ҮзӮ№еҮ» еҸҜиҒҡз„ҰгҖӮ
    • Qt::StrongFocus вҖ”вҖ” Tab й”® + йј ж ҮзӮ№еҮ» еқҮеҸҜиҒҡз„ҰпјҲй»ҳи®ӨпјүгҖӮ
    • Qt::WheelFocus вҖ”вҖ” еҗҢ StrongFocusпјҢйўқеӨ–ж”ҜжҢҒ ж»ҡиҪ®иҒҡз„ҰпјҲжһҒе°‘з”ЁпјүгҖӮ
