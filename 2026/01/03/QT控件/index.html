QWidget的enable属性

API 说明
isEnabled() 获取到控件的可⽤状态.
setEnabled 设置控件是否可使⽤. true 表⽰可⽤, false 表⽰禁⽤.
  • 所谓”禁⽤”指的是该控件不能接收任何⽤⼾的输⼊事件,并且外观上往往是灰⾊的.
  • 如果⼀个widget被禁⽤,则该widget的⼦元素也被禁⽤.
